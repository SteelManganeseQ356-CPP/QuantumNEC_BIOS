ubuntu需要如下操作
安装GCC: sudo apt-get install gcc
安装git: sudo apt-get install git
安装vim: sudo apt-get install vim
下载nasm源码并安装
下载bochs源码并安装
安装re2c: sudo apt install re2c
安装ninja: sudo apt-get install ninja-build
下载qemu源码并安装

ubuntu安装ssh: sudo apt-get install ssh

Host xxx
	HostName 192.168.xxx.xxx
	User xxx
Host->远程连接的名字
HostName->远程连接的机子的IP地址
User->远程连接的机子的用户名
